Student Performance Prediction ETL Pipeline

Role: Lead Data Engineer and Analyst Date: March 2024 Context: Database Systems and Predictive Analytics
Python Pandas Scikit Learn ETL

Problem

Academic data is often fragmented across multiple systems. The objective was to build a unified ETL pipeline capable of integrating academic, demographic, and departmental data to predict student performance.

Approach

  • Extracted data from multiple simulated enterprise sources
  • Performed data validation, sparsity filtering, and outlier removal
  • Merged datasets into a master analytical table using relational keys
  • Trained both global and individualized regression models

Results

  • R squared of 0.94 explaining student grade variance
  • Strong linear relationship between effort and outcome
  • Demonstrated feasibility of personalized academic forecasting
