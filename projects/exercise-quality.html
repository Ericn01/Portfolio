Project

Exercise Quality Prediction with Wearable Sensors

Role: Lead Data Analyst Date: January 2026 Context: Machine Learning and Motion Analysis
R Random Forest Caret IoT Data

Problem

Detecting incorrect movement patterns is critical in health and industrial environments. This project focused on building a classifier capable of identifying exercise execution errors using high frequency wearable sensor data.

Approach

  • Processed approximately 20,000 observations from four sensor locations
  • Removed near zero variance and highly sparse features to improve model stability
  • Trained a Random Forest ensemble to capture non linear motion patterns
  • Evaluated performance using a 75 25 train test split

Results

  • Over 99 percent classification accuracy
  • Near perfect agreement measured via Kappa statistic
  • Identified belt and dumbbell sensors as dominant signal sources
