Project Description

Dynamic Javascript website that retrieves data from over 25,000 cities. This includes population data, weather data, current time, along with a short description of the location.

API's Used

  • Google Maps
    • Places API
    • Dynamic Map API
  • Wikipedia's PHP API
  • OpenWeather API
